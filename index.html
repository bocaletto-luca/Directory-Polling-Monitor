Directory Polling Monitor

Pure-Python, polling-based file & folder watcher. No external libraries required.

GitHub Repository

Overview

Directory Polling Monitor is a single-file Python script (stdlib only) that lets you watch multiple directories for changes—creations, deletions, modifications— via configurable polling intervals. Press ESC at any time to stop monitoring and return to the menu.

Key Features

  • Monitor one or more directories simultaneously
  • Recursive or non-recursive scan modes
  • Include or exclude hidden files and folders
  • Advanced glob filters for include/exclude patterns
  • Real-time logging to console and/or file
  • Interactive text-menu interface
  • ESC key to interrupt and return to configuration
  • Zero external dependencies (Python 3.6+ stdlib only)

Installation

# clone or download
git clone https://github.com/bocaletto-luca/directory-polling-monitor.git
cd directory-polling-monitor

# (optional) make executable
chmod +x monitor_poll_menu.py

Requires Python 3.6 or newer.

Usage

python3 monitor_poll_menu.py
# or, if executable:
./monitor_poll_menu.py

Use the interactive menu to:
1) Add/remove directories
2) Toggle recursion and hidden entries
3) Set polling interval and log file
4) Define include/exclude glob filters
5) Start or stop monitoring

Advanced Filters

In the menu’s Advanced filters submenu, you can manage INCLUDE and EXCLUDE patterns using Unix-style globs:

  • *.log to include only log files
  • data/**/*.csv to include CSVs in any subfolder
  • temp/* to exclude everything under temp/

How ESC Works

During monitoring, the script temporarily switches the terminal to “cbreak” mode, allowing single-key detection. Pressing the ESC key immediately breaks the loop, restores the terminal to normal mode, and returns you to the main menu.

Logging Format

2025-07-05 22:58:53,080 INFO     [ /path/to/dir ] +Added   FILE: example.txt
2025-07-05 22:59:12,345 INFO     [ /path/to/dir ] *Modified DIR : docs/
2025-07-05 22:59:20,123 INFO     [ /path/to/dir ] -Removed  FILE: old.log

+Added = creation
*Modified = timestamp changed
-Removed = deletion
